v-grid.js

Version 1.0.0
Github
Place anything you want vertically using a grid layout.
** Images from Google Images

<Usage>

In your HTML code, surround your target nodes with a parent node. Declare an 'id' for parent node and 'class' for target node. Just like the code below. <div id="parent_node">
  <img class="target_node" src="photo.jpg" />
  <img class="target_node" src="photo2.jpg" />
  ...
</div> Then include the following in your project. // CSS, declare the following at your own CSS file.
#parent_node{
  position: relative;
  overflow: hidden;
}

#target_node{
  position: absolute;
  overflow: hidden;
}

// JS
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.2/jquery.min.js"></script> // skip this if you have it already
<script src="script2-min.js"></script> In your window.onload function call, add a 'vgrid' function call like below. $('#parent_node').vgrid('.target_node');
 ...and you're done!

Options

Options : Type Default Description
gap : number 10 Gap between item, and it is measured in pixel. Just declare it without 'px'.
item_per_row: number 6 The number of item per row. Width of each item will be calculated according to it.
Here's how to use them. $('#parent_node').vgrid('.target_node',{
  gap : 10,
  item_per_row : 4
});
